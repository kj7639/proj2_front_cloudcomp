## Getting Started 

First, run the development server and dependencies:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
 
## INSTALL THESE PACKAGES PLS

```bash
npm install next-auth@beta
# and
npm install otplib qrcode
# and
npm i better-auth
# and
npm i prisma @prisma/client @prisma/adapter-pg pg
npx prisma init
# and
npm install --save-dev prisma dotenv
# and
npx prisma generate
npx prisma db push
# and
npm i @prisma/adapter-pg pg
# and
npx @better-auth/cli@latest generate
# and
npx @better-auth/cli@latest generate --with-plugins=twoFactor
# and maybe
npm install next-auth/react

# handle deprecation
npm install -g npm@11
npm install rimraf@latest glob@10 eslint@latest

npm install --save-dev eslint@^9.0.0
```
